## Репозиторий для заданий

1. HTML/CSS

2. JavaScript

3. Архитектура

## Требования

- Node.js версии `>= 18.*`
- Пакетный менеджер **npm**

## Запуск проекта

1. Используйте команду `npm install`, чтобы установить все необходимые зависимости

2. Для запуска проекта в режиме разработки используйте команду `npm run dev`
